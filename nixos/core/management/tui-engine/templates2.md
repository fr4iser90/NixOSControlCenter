## **Layout-Algorithmen:**

### **Horizontal Stack:**
```go
body := lipgloss.JoinHorizontal(lipgloss.Top, panel1, panel2, panel3)
```

### **Vertical Stack:**  
```go
body := lipgloss.JoinVertical(lipgloss.Left, panel1, panel2, panel3)
```

### **Grid (2x2):**
```go
row1 := lipgloss.JoinHorizontal(lipgloss.Top, p1, p2)
row2 := lipgloss.JoinHorizontal(lipgloss.Top, p3, p4)
body := lipgloss.JoinVertical(lipgloss.Left, row1, row2)
```

### **Sidebar Links:**
```go
sidebar := renderSidebar()
main := renderMainContent() 
body := lipgloss.JoinHorizontal(lipgloss.Top, sidebar, main)
```
